Overview

This provides a variant of the rollup_bundle rule that works better for Angular apps.

It registers @angular-devkit/build-optimizer as a rollup plugin, to get better optimization. It also uses ESM5 format inputs, as this is what build-optimizer is hard-coded to look for and transform.

ng_rollup_bundle

ng_rollup_bundle(name, deps)

Run Rollup with the Build Optimizer plugin.

This rule extends from the rollup_bundle rule, so attributes and outputs of that rule are used here too.

Attributes

name

Name; Required

A unique name for this rule.
deps

List of labels; Optional; Default is []

Other targets that provide JavaScript files. Typically this will be ts_library or ng_module targets.
